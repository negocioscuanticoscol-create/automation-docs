Claude Code + MCP
nuevo
Constructor inteligente de flujos n8n. Describes en lenguaje natural y Claude crea el workflow automáticamente via MCP.
  • Crear flujos sin tocar nodos
  • Adaptar templates existentes
  • Acelerar replicación por negocio
¿Qué es MCP? Model Context Protocol — un puente que permite a Claude operar herramientas externas directamente. Es como una API, pero Claude decide cuándo y cómo usarla sin que tú escribas el código de conexión.
